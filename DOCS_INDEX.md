## üìû Getting Help

### If You're Stuck

1. **Check the docs** - Is it in one of these files?
2. **Search the code** - Is there similar code that does what you need?
3. **Review examples** - Look at working components in `/src/`
4. **Read error messages** - They often tell you exactly what's wrong
5. **Check the specific doc** - If about X, find X in the docs

### Asking AI Models for Help

When asking Claude, ChatGPT, or another AI model:

```
I'm working on [PROJECT: Teacher Lesson Planning Portal].
I need to [describe the task].
I've read [which docs: README, ARCHITECTURE, etc.].
The error/issue is: [exact error or description].
I've already tried: [what you've attempted].

Can you help with [specific question]?
See [relevant doc] for context.
```

This helps the model understand your context quickly.
